# cisco-network-design
This is a core project for Networking Principles (NET1014). Acting as a Network Engineer for Sunway University, the primary objective is to design and simulate a segmented network connecting 4 labs. This includes a full OSI analysis and functional Cisco Packet Tracer (.pkt) implementation.

🤝 This project was a collaborative effort, completed by a team of 5 members, with my role focused on leadership and technical implementation.

💻 Technical Scope for the project:
OSI Layer Analysis: Conducted a deep dive into the 7-layer stack, evaluating protocols and their real-world applications.
Network Segmentation: Implemented a logical addressing scheme using the 192.168.10.0/24 network.
Subnetting Strategy: Optimized IP allocation for 34+ hosts across two separate buildings to ensure security and management efficiency.
Simulation: Built and validated a functional virtual network using Cisco Packet Tracer (.pkt).
